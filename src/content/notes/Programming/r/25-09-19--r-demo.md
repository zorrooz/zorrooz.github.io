# DESeq2 + ggplot2 绘制火山图

核心代码片段：

```r
library(ggplot2)
ggplot(res, aes(x=log2FoldChange, y=-log10(padj))) +
  geom_point(aes(color=significant)) +
  theme_minimal()
```
