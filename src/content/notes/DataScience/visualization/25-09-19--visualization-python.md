# Python 数据可视化工具

## Matplotlib 基础

```python
import matplotlib.pyplot as plt

# 创建简单图表
plt.plot(x, y)
plt.xlabel('X轴')
plt.ylabel('Y轴')
plt.title('标题')
plt.show()
```

## 库比较

- **Matplotlib**: 基础绘图库
- **Seaborn**: 统计图表，基于Matplotlib
- **Plotly**: 交互式图表
