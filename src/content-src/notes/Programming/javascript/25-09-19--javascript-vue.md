# Vue.js 前端开发入门

## 基本语法

```javascript
// Vue 组件示例
export default {
  data() {
    return {
      message: 'Hello Vue!'
    }
  },
  methods: {
    updateMessage() {
      this.message = 'Updated!'
    }
  }
}
```

## 组件通信

- Props：父组件向子组件传递数据
- Events：子组件向父组件发送消息
- Vuex：全局状态管理
