# 实验方法

**以下内容皆为虚构！**

## 1. 样品制备

### 蛋白质表达与纯化
```bash
# 表达系统
宿主：E. coli BL21(DE3)
诱导条件：0.5 mM IPTG, 18°C, 16小时

# 纯化步骤
1. Ni-NTA 亲和层析
2. 凝胶过滤层析 (Superdex 200)
3. 浓缩至 3 mg/mL
```

### 负染电镜筛选
- 染色剂：2% 醋酸铀酰
- 数据采集：200 kV, 50,000x 放大倍数
- 筛选标准：单分散、完整颗粒

## 2. 冷冻电镜数据采集

### 样品制备
- 3 μL 样品滴加在 Quantifoil R1.2/1.3 网格
- 滤纸吸干，液氮快速冷冻
- 工作温度：-180°C

### 采集参数
| 参数 | 值 |
|------|-----|
| 加速电压 | 300 kV |
| 像素尺寸 | 1.5 Å |
| 电子剂量 | 50 e⁻/Å² |
| 帧数 | 50 frames |

## 3. 数据处理

### 运动矫正
- 软件：MotionCor2
- 修正帧间运动和漂移

### 颗粒挑选
- 方法：模板匹配
- 软件：RELION 3.1
- 颗粒数量：~50,000

### 二维分类
- 类别数：20
- 选择标准：清晰特征

## 4. 三维重建

### 初始模型
- 方法：随机圆锥重构
- 分辨率：~8 Å

### 精细重建
- 迭代：20 轮
- 分辨率：3.0 Å
- 分辨率标准：FSC 0.143

## 5. 模型构建与优化

### 初始模型搭建
- 软件：Coot
- 方法：自动建模 + 手动修正

### 结构优化
- 软件：Phenix.refine
- 优化内容：
  - 原子坐标
  - 温度因子
  - 占有率

### 质量验证
- Ramachandran plot：96.5% 优势区域
- 几何偏差：键长 0.01 Å，键角 1.2°
- MolProbity score：1.8
